Hal-hal yang dipelajari :
1. data cleaning di section "transform data" (untuk membersihkan/standarisasi data)
2. join dan data modelling (join untuk memakai lebih dari 1 dataset dengan kolom key yang sama, modelling untuk menjadikan visualisasi kita konsisten)
3. membuat hierarcy (drill down) - (untuk membuat detail suatu data. contoh tahun 2024 di detailkan menjadi 12 bulan, masing masing bulan bisa di detailkan menjadi 30 hari +-)
4. membuat folder untuk kolom (missal kolom 'math score', 'math tutor' ditaruh menjadi 1 folder biar rapi dan gampang di cari)
5. membuat DAX (Data Analyst Expression) measure (untuk membuat kolom perhitungan secara manual. Contoh = ada kolom Harga, dan kolom biaya pembuatan. Kita bisa bikin kolom baru 'keuntungan' dengan pakai measure baru (kolom Harga - kolom biaya pembuatan))
6. membuat quick measure (mempercepat dalam membuat measure yang rumit)
7. Visualisasi :
- Exclude = mengecualikan data dengan kategori tertentu (contoh : siswa yang tidak memiliki value di kolom ekstrakulikuler di exclude saat membuat visualisasi data ekstrakulikuler)
- Data label = menyantumkan nilai di visualisasinya (biar gampang tau nilainya)
- Themes = template warna visualisasi dari power BI
- Customize themes = buat tema sesuai yg di inginkan
- Canvas background = buat ngasih background gambar maupun warna
- insert section = buat kasih judul, gambar, tombol, dll
- picture button = jadiin gambar sebagai button pake fitur action di format
- format pointer = kopi paste format visualisasi ke visualisasi lain (contohnya kaya kopi paste setingan transparansi background)
- Conditional formatting = menandai value mana yang masuk kriteria (contoh value lebih dari 10k diwarnai hijau, makin sedikit warnanya makin merah)
- Jenis visualisasi = Jenis visualisasi bisa di tambah dengan cara 'get more' di bagian titik 3 di list visualisasi
- Q&A Visualization = visualisasi dibuatin AI berdasarkan data kita, cek di list visualisasi yang disediain
- tooltip page = buat costum tooltip di visual yang di inginkan
